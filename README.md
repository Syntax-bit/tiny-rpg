# Tiny RPG

Tiny RPG is a stylized action framework built in Unity 6 and C#, heavily inspired by the core gameplay mechanics of **World of Warcraft**. 

This project is a deliberate technical playground designed to master **clean code**, **decoupled systems**, and **scalable design patterns** in game development.

> 🚧 **Project Status:** Active Development

---

## 🗺️ Feature Roadmap

The goal is to recreate the core loops of a classic MMORPG, focusing on clean data handling and modular systems.

### ⚔️ Combat & AI
* [x] **Tab-Targeting & Selection:** WoW-style target cycling and raycast mouse selection.
* [ ] **Ability Framework:** Universal casting loops sharing logic between players and AI.
* [ ] **Enemy AI:** Simple behavior states (Patrol, Aggro, and Combat).

### 🎒 Progression & Economy
* [ ] **Talent Trees:** Unlockable nodes that modify base ability stats.
* [ ] **Inventory System:** Grid-based item management and data handling.
* [ ] **World NPCs:** Interactive merchants (Traders and Blacksmiths) for buying, selling, and repairs.

---

## 🎓 Learning Objectives

* **Clean Code:** Writing scalable, readable, and self-documenting C#.
* **Decoupled Architecture:** Minimizing dependencies to prevent "spaghetti code".
* **Memory Management:** Managing data safely to avoid memory leaks with persistent assets.
* **Modular Design:** Building systems that are easy to expand down the road.
